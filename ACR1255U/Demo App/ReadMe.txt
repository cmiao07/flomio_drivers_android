ACS Bluetooth Android Demo 
Advanced Card Systems Ltd.

Introduction
------------

The ACS BTDemo app was designed to be used with ACR3901U-S1 and ACR1255U-J1 Bluetooth Card Readers.


Installation
------------

1. Copy the "BTDemo.apk" to your Android device. 
2. Launch the installation by clicking the file icon using any File Explorer application.
